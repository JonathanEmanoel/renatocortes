# Assets

Pastas reservadas para midias futuras do projeto.

- `logos/`: logo principal, variacoes e simbolos.
- `backgrounds/`: fotos de fundo e imagens de hero.
- `products/`: imagens dos produtos da loja.
- `barbers/`: fotos dos barbeiros e equipe.
- `icons/`: icones customizados fora do Lucide.
- `banners/`: banners promocionais e assinaturas.
